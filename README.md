# PCSAPO 飲食店導入パッケージ 営業支援Webアプリ

PCSAPO マキシ企画が飲食店向け営業で使用する、提案・ヒアリング・見積・契約・保守・引渡しを一元管理する静的Webアプリです。既存のA4縦22ページ営業資料の構成と、ネイビー・オレンジ・ホワイトを基調としたデザインを踏襲しています。

## お客様向け・管理者用

- お客様向けでは、サービス詳細、PCSAPOの役割、ヒアリング、導入状況、FAQ、操作マニュアルを表示します。
- お客様向けのヒアリングは「サービス選択 → ご希望入力 → 内容確認 → 受付方法」の4ステップで進みます。
- 受付方法は端末の共有機能、メール、LINE、PDF・紙から選択できます。現行版は自動受付ではなく、PCSAPOから受付確認の返信が届いた時点で受付完了です。
- 管理者用では、ダッシュボード、案件テンプレート、チェック、見積、契約、引渡し、PDF・印刷を扱います。
- 管理者用は端末ごとに設定する簡易パスコードで保護します。静的なGitHub Pages上の機能であり、サーバー認証ではありません。

## 料金マスター

- `PCSAPO作業料金`: 制作・設定・運用支援として見積へ追加できます。
- `公式料金・機器`: STUDIO.Design、Square、funfo各社への直接支払いとして参考表示し、PCSAPOの見積合計には入りません。
- `販促実行支援`: 施設調査、設置確認、訪問・納品、補充・報告等を見積へ追加できます。
- キーワード、カテゴリー、料金種別で絞り込み、Squareやfunfo等の対象項目だけを表示できます。

案件、署名、料金マスターはブラウザの `localStorage` に保存され、別端末へは自動同期されません。共有端末では利用後に案件データを削除してください。

## 資料・公式リンク管理

- 管理者画面で「共通配布資料」「案件別資料」「公式リンク」の3種類を管理できます。
- 表示名、説明、対象サービス、カテゴリー、URL、確認日、表示順、QR画像を追加・編集できます。
- お客様への表示、有効・無効、主要リンク、アーカイブを切り替えられます。
- お客様画面には、選択中サービスに関係する公開資料と公式リンクだけを表示します。
- 設定はJSONで書き出し・読み込みできます。

現段階の管理設定は端末内の `localStorage` に保存されます。そのため、管理者が変更した内容を別端末のお客様へ即時配信するクラウドCMSではありません。PDF本体はGitHubへ追加済みの相対パス、または公開済みURLを登録してください。

## 主な機能

- PC、iPad、iPhone、Android対応のレスポンシブUI
- ヒアリング、導入チェック、定期点検などの入力フォーム
- サービスごとの基本内容と、ヒアリング内オプションの分離表示
- STUDIO.Design・Square・funfoをお客様名義で登録し、PCSAPOを編集者・スタッフとして招待する安全な導入フロー
- カード情報・本人確認書類・銀行口座・ID・パスワードをアプリへ保存しないプライバシー確認
- サービスごとのアカウント準備状況と、納品時の権限見直しチェック
- ヒアリングで選択したオプションを概算見積へ即時反映
- 印刷物・販促実行支援は仕様と作業範囲を選択して個別積算
- 料金明細、数量、小計、消費税、合計の自動計算
- オプションを複数選択し、数量付きで見積へ直接追加
- 一覧にないオプションの名称・数量・単価を自由入力
- Canvas電子署名（Apple Pencil、マウス、指に対応）
- 契約番号の自動採番、契約・サブスク・引渡し情報の管理
- LocalStorageによる案件の保存、編集、削除、複製
- 案件一覧のCSV出力・取込
- ブラウザ印刷によるA4縦PDF保存
- ヒアリングで選択したサービスだけを対象に、事前準備・公式登録・制作工程・概算見積を個別または一括でプレビュー
- 選択サービスごとの「事前確認資料」と「概算見積書」をA4縦PDFとして保存・印刷
- 事前確認資料にPCSAPO補助案内、各社公式手順リンク、印刷用QRコード、公式情報確認日を表示
- 選択した複数サービスをまとめた「合計概算見積書」をプレビュー・PDF保存・印刷
- サービス説明ページは登録案内に限定し、実際の確認チェックはヒアリング画面へ集約
- 管理者による共通配布資料・案件別資料・公式リンクの追加、編集、公開切替、アーカイブ
- お客様向けのサービス別資料・公式リンク一覧
- Web Share API、LINE、メール共有
- PWA、オフライン閲覧、ホーム画面追加
- ロゴをクリックして入口または各モードのホームへ移動
- 導入・保守見積書のPCSAPO印「あり／なし」切替
- funfoオンライン決済の申請条件・公式審査目安（1〜2か月）を店頭会計と分けて表示
- 補助金・支援制度は無料の公開情報案内として記録し、見積合計には加算しない
- 外部ライブラリ、ビルド作業、サーバー不要

## ローカルで確認

Service Workerを含むPWA機能は `file://` では動作しないため、任意の静的Webサーバーを使用してください。

```sh
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080/pcsapo-sales-app/` を開きます。

## GitHub Pagesで公開

1. このフォルダの内容をGitHubリポジトリへ追加します。
2. GitHubの **Settings > Pages** を開きます。
3. **Deploy from a branch** を選択します。
4. 公開対象のブランチと、配置に合わせて `/ (root)` または `/docs` を選択します。

すべて相対パスのため、ユーザーサイトとプロジェクトサイトのどちらでも動作します。HTTPS環境ではPWAと共有APIを利用できます。

## データについて

案件データと署名画像は、使用中のブラウザのLocalStorageに保存されます。クラウドや他端末には自動同期されません。ブラウザデータを削除すると案件も削除されるため、案件一覧から定期的にCSVを出力してください。

各社サービスのID・パスワード、カード番号、本人確認書類、銀行口座情報は、本アプリへ入力・保存しない運用です。お客様が各社公式画面へ直接登録し、PCSAPOには必要最小限の編集・スタッフ権限を付与してください。

管理者用の簡易パスコードも端末ごとに保存されます。忘れた場合は管理者ログイン画面の「この端末のパスコードをリセット」を使用できます。案件データと料金マスターは削除されません。

CSV取込は案件一覧の基本情報を復元します。詳細フォーム、署名、見積明細を含む完全バックアップではありません。

## PDF保存

お客様向け資料は「ヒアリング／概算見積」でサービスを選択し、「確認資料・PDF」から個別または一括プレビューします。「PDF保存・印刷」を押し、ブラウザの印刷画面で次を指定します。

管理者用の契約書・見積書等は「PDF・印刷」画面から出力します。

- 用紙: A4
- 向き: 縦
- 送信先: PDFに保存
- 背景グラフィック: オン推奨
