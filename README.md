# PCSAPO 飲食店導入パッケージ 営業支援Webアプリ

PCSAPO マキシ企画が飲食店向け営業で使用する、提案・ヒアリング・見積・契約・保守・引渡しを一元管理する静的Webアプリです。既存のA4縦22ページ営業資料の構成と、ネイビー・オレンジ・ホワイトを基調としたデザインを踏襲しています。

## 主な機能

- PC、iPad、iPhone、Android対応のレスポンシブUI
- ヒアリング、導入チェック、定期点検などの入力フォーム
- 料金明細、数量、小計、消費税、合計の自動計算
- オプションを複数選択し、数量付きで見積へ直接追加
- 一覧にないオプションの名称・数量・単価を自由入力
- Canvas電子署名（Apple Pencil、マウス、指に対応）
- 契約番号の自動採番、契約・サブスク・引渡し情報の管理
- LocalStorageによる案件の保存、編集、削除、複製
- 案件一覧のCSV出力・取込
- ブラウザ印刷によるA4縦PDF保存
- Web Share API、LINE、メール共有
- PWA、オフライン閲覧、ホーム画面追加
- 外部ライブラリ、ビルド作業、サーバー不要

## ローカルで確認

Service Workerを含むPWA機能は `file://` では動作しないため、任意の静的Webサーバーを使用してください。

```sh
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080/pcsapo-sales-app/` を開きます。

## GitHub Pagesで公開

1. このフォルダの内容をGitHubリポジトリへ追加します。
2. GitHubの **Settings > Pages** を開きます。
3. **Deploy from a branch** を選択します。
4. 公開対象のブランチと、配置に合わせて `/ (root)` または `/docs` を選択します。

すべて相対パスのため、ユーザーサイトとプロジェクトサイトのどちらでも動作します。HTTPS環境ではPWAと共有APIを利用できます。

## データについて

案件データと署名画像は、使用中のブラウザのLocalStorageに保存されます。クラウドや他端末には自動同期されません。ブラウザデータを削除すると案件も削除されるため、案件一覧から定期的にCSVを出力してください。

CSV取込は案件一覧の基本情報を復元します。詳細フォーム、署名、見積明細を含む完全バックアップではありません。

## PDF保存

「PDF・印刷」画面から「PDF保存・印刷」を押し、ブラウザの印刷画面で次を指定します。

- 用紙: A4
- 向き: 縦
- 送信先: PDFに保存
- 背景グラフィック: オン推奨
